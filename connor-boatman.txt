	1. What is Semantic HTML?

    7. What is an id? 
        An id is an attribute that can only be used on one element once.
    
    8. What elements can I add an id to? 
        In HTML5 an id can be used on any element, in HTML4 an id cannot be used on any of the following elements: <base>, <head>, <html>, <meta>, <param>, <script>, <style>, and <title>.

    10. What is a class?

        A class can be used multiple times whereas an id can only be used once. You could use a class tag to mark different elements of the HTML you want to apply the same css attributes to and add an id tag to be able to select just once specific element in the HTML if you wanted to change it from the other elements of the same type.
    
    20. How do we select classes in CSS?
        To select a class in css you use '.class{}'.
    
    21. How do we select a p element with a single class of “human””?

    22. What is a parent child selector? When would this be useful? 
        A parent child selector is where you can select a specific tag inside of another tag, so if you wanted to select all 'p' tags inside of a 'div' it would only affect the 'p' tags inside that 'div' instead of all of the 'p' tags in the HTML.
    
    23. How do you select all links within a div with the class of sidebar?

    24. What is a pseudo selector?
        A pseudo selector is a tag that is added to a selector to specify a change under special circumstances like ':hover' changes the element when the mouse hovers over the specified element.
    
    25. What do we use the change the spacing between lines?

    26. What do we use to change the spacing between letters?

    27. What do we use to to change everything to CAPITALS? lowercase? Capitalize?
        To change text to all capitals, lowercase, or capitalized we use the 'text-transform' property. For all capitals it is 'text-transform: uppercase', for all lowercase it is 'text-transform: lowercase', and for standard capitalization it is 'text-transform: capitalize'.
    
    28. How do I add a 1px border around my div that is dotted and black?

    37. Walk me through Lambda's git flow. 
        Lambda's git flow is as follows:
            -Fork the repo
            -Clone your forked repo to your personal computer
            -Create a new branch in the command line using the command:
                $ git checkout -b 'firstname-lastname'
            -Work on your project in your text editor of choice
            -Add and commit changes to your branch in the command line using the commands:
                $ git add .
                $ git commit -m "message detailing changes"
            -Push the changes to the repo on GitHub in the command line using the command:
                $ git push -u origin branch-name
            -Submit a pull request on your forked repo on the GitHub webpage
            -Make sure before submitting a pull request that the base repository listed is your personal branch
            -Submit pull request link to Canvas
            -Do not merge your own pull request
