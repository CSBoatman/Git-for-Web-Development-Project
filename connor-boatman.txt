    7. What is an id? 
        An id is an attribute that can only be used on one element once.
    
    8. What elements can I add an id to? 
        In HTML5 an id can be used on any element, in HTML4 an id cannot be used on any of the following elements: <base>, <head>, <html>, <meta>, <param>, <script>, <style>, and <title>.
    
    9. How many times can I use the same id on a page? 
        An id can only be used once.
    
    10. What is a class? 
        A class is an attribute the can be used multiple times on multiple elements.
    
    11. What elements can I add a class to? 
        Classes can be used on any element.
    
    12. How many times can I use the same class on a page? 
        As many times as you'd like, though using the same class multiple times can cause confusion or problems.
    
    13. How do I get my link to open in a new tab?

    15. How do I reference an id?
